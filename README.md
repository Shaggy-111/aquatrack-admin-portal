# AquaTrack Admin Portal

## Overview
AquaTrack Admin Portal is a web-based dashboard designed to manage customers, orders, vendors, and delivery operations for the AquaTrack system.

## Features
- Admin dashboard and analytics
- Order management
- Customer and vendor control
- Role-based access
- Real-time data handling
- Clean and structured UI

## Tech Stack
- Web Portal / Dashboard
- Backend API integration
- Database connectivity
- Role-based system

## Screenshots
See the /screenshots folder for portal preview.

## Use Case
Used by administrators to manage the full water delivery workflow including orders, customers, and operational data.
